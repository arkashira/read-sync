# Reading Position Synchronization

This project provides a simple reading position synchronization service.

## Usage

1. Create a `ReadingPositionService` instance.
2. Save a reading position using `save_position`.
3. Get the current reading position using `get_position`.
4. Push a new reading position using `push_position`.
5. Pull the latest reading position using `pull_position`.
6. Resolve conflicts using `resolve_conflict`.

## Testing

Run the tests using `pytest`.
