# Business Model Canvas

## Customer Segments
- E-book enthusiasts reading across multiple devices (e-readers, phones, tablets)
- Privacy-conscious users avoiding commercial platforms with data collection
- Technical users comfortable with self-hosted solutions
- Existing e-reader owners frustrated with limited sync options
- Small organizations needing controlled document distribution

## Value Propositions
- Seamless reading position synchronization across all devices without third-party servers
- Ad-free reading experience with complete user control over data
- Cross-platform compatibility supporting major e-reader formats and devices
- Self-hosted architecture ensuring privacy and data ownership
- No vendor lock-in with open-source foundation

## Channels
- Direct downloads through GitHub repository and official website
- Digital marketplaces specializing in privacy-focused applications
- Online communities focused on e-books and digital privacy
- Technical forums and platforms (Reddit, specialized tech sites)
- Partnerships with e-reader manufacturers for pre-installation

## Customer Relationships
- Self-service model with comprehensive documentation and tutorials
- Community forums for peer-to-peer support and feature requests
- Optional premium support contracts for enterprise users
- Regular updates based on user feedback and market demands
- Direct communication channels through GitHub issues and community platforms

## Revenue Streams
- One-time purchase fee for premium features ($29.99 USD / ~1,000 THB)
- Optional cloud sync service for non-self-hosted users ($4.99 USD / ~165 THB monthly)
- Enterprise licenses for organizations with advanced features ($99 USD / ~3,300 THB annually)
- Optional premium support packages ($9.99 USD / ~330 THB monthly)
- White-label licensing for e-reader manufacturers ($15-25 USD per unit)

## Key Resources
- Development team specializing in cross-platform applications and sync technologies
- Infrastructure for optional cloud sync service
- Documentation and knowledge base resources
- GitHub repository and community management
- Security expertise ensuring user data protection

## Key Activities
- Software development and maintenance across multiple platforms
- Security audits and regular updates to protect user data
- User support and community management
- Marketing to target privacy-conscious technical users
- Partnerships with e-reader manufacturers and content platforms

## Key Partners
- E-reader manufacturers for compatibility and pre-installation opportunities
- Bookstores and digital content providers for integration
- Cloud hosting providers for optional sync service
- Privacy advocacy groups for promotion and credibility
- Technical communities and influencers for adoption and feedback

## Cost Structure
- Development team salaries (primary cost at ~70% of expenses)
- Infrastructure costs for optional cloud sync service (~15%)
- Marketing and customer acquisition (~8%)
- Legal and compliance costs (~4%)
- Security audits and maintenance (~3%)
