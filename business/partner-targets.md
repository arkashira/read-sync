# partner-targets.md

## Partner Integration Roadmap

### 1. Project Gutenberg
- **Free-tier**: All content is free (no limits)
- **Integration effort**: S (simple API, well-documented)
- **Value-add**: Immediate access to 70,000+ free public domain books, solving the "what to read" problem
- **Revenue potential**: None (all free content)
- **Priority**: High - provides immediate content value

### 2. Calibre
- **Free-tier**: Free open-source software
- **Integration effort**: M (requires library management integration)
- **Value-add**: Enables users to import existing e-book libraries, solving the "I own books but can't access them" problem
- **Revenue potential**: None (open-source)
- **Priority**: High - addresses core user need

### 3. Dropbox
- **Free-tier**: 2GB storage
- **Integration effort**: M (standard API integration)
- **Value-add**: Cloud storage integration for personal e-books, solving the "how to store my books" problem
- **Revenue potential**: Medium - affiliate revenue for premium upgrades (15-20% commission)
- **Priority**: Medium - practical utility with revenue upside

### 4. Goodreads
- **Free-tier**: Free service with optional premium features
- **Integration effort**: M (API available with authentication requirements)
- **Value-add**: Social features (reviews, ratings, reading challenges), solving the "book discovery and community" problem
- **Revenue potential**: Medium - affiliate revenue for book purchases (6-8% commission)
- **Priority**: Medium - enhances user engagement

### 5. Amazon Kindle
- **Free-tier**: Free access to purchased books
- **Integration effort**: L (complex due to proprietary formats and DRM)
- **Value-add**: Access to users' existing Kindle libraries, solving the "I've already bought books" problem
- **Revenue potential**: High - affiliate revenue for book purchases (up to 10%)
- **Priority**: High - addresses major market segment

### 6. Evernote
- **Free-tier**: 60MB/month upload limit
- **Integration effort**: M (API available)
- **Value-add**: Sync reading notes and highlights, solving the "how to remember what I read" problem
- **Revenue potential**: Low - affiliate revenue for premium upgrades (5-7%)
- **Priority**: Medium - enhances reading experience

### 7. Docker
- **Free-tier**: Free community edition
- **Integration effort**: S (simple containerization)
- **Value-add**: Easy deployment for technically inclined users, solving the "how to set this up" problem
- **Revenue potential**: None (open-source)
- **Priority**: Medium - improves accessibility for power users

### Implementation Timeline
- **Phase 1 (0-3 months)**: Project Gutenberg, Calibre, Docker
- **Phase 2 (3-6 months)**: Dropbox, Goodreads
- **Phase 3 (6-9 months)**: Amazon Kindle, Evernote
